Node Scraper
===
A simple web scraper to extract Event listings from WeGotTickets.com

Dependencies
---
Dependencies are given in package.json

In addition to Node (0.10.26 tested)

    "request": "latest",
    "cheerio": "latest",
    "fs": "latest"

To run
---
Open terminal at folder

    npm install
    ...
    node main.js
